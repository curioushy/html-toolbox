# HTML Toolbox

A collection of standalone, single-file HTML tools. Each file is fully self-contained with zero dependencies — just open in a browser.

## Tools

| Tool | Description |
|------|-------------|
| [text-compare.html](text-compare.html) | Side-by-side text comparison with word-level and character-level diff, inline/side-by-side views, and Outlook-compatible rich copy |

## Features

- **Zero dependencies** — no frameworks, no build steps, no server needed
- **Fully local** — nothing leaves your machine
- **Portable** — each tool is a single `.html` file
